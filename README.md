# Homebrew Tap for LidLock

Install LidLock with Homebrew:

```sh
brew install --cask DelPage/lidlock/lidlock
```

Or tap first:

```sh
brew tap DelPage/lidlock
brew install --cask lidlock
```

LidLock is a signed and notarized macOS app that keeps your MacBook running with
the lid closed for servers, downloads, bots, AI work, and long builds.

- Product page: https://delpageinc.com/lidlock
- Public repo and releases: https://github.com/DelPage/lidlock
- Privacy policy: https://delpageinc.com/privacy/lidlock

The cask installs the signed/notarized `LidLock.dmg` from the public GitHub
release.
